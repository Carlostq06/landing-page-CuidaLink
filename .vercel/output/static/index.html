Preguntas Frecuentes

Resolvemos tus dudas principales para que empieces con confianza.

Necesito un dispositivo especial?expand_more
No. CuidaLink funciona directamente desde cualquier smartphone Android para el paciente y desde cualquier navegador web para el cuidador. No requiere comprar hardware adicional ni relojes costosos.
Como se protegen los datos?expand_more
Utilizamos encriptacion SSL y la infraestructura de seguridad de Supabase para asegurar que solo tu tengas acceso a la ubicacion y datos medicos de tu familiar.
Funciona sin internet?expand_more
La aplicacion requiere una conexion (Wi-Fi o datos moviles) para enviar alertas en tiempo real y actualizar el GPS. Sin embargo, cuenta con un sistema de ultima ubicacion conocida para que siempre sepas donde estuvo por ultima vez.
